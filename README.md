# My Data Science Notebook

## Description
This repository contains a beginner-friendly Jupyter Notebook that introduces fundamental concepts in Data Science.

## Contents
- Data Science languages: Python, R, SQL, Julia  
- Data Science libraries: Pandas, NumPy, Matplotlib, Scikit-learn  
- Data Science tools: Jupyter Notebook, GitHub, Tableau, RStudio  
- Basic Python arithmetic operations  
- Simple unit conversion example (minutes to hours)

## Objectives
- Understand basic Python syntax and operations  
- Learn commonly used data science tools and libraries  
- Practice working with Jupyter Notebooks  
- Perform simple calculations and transformations
